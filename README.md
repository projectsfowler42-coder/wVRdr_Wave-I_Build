# wVRdr_Wave-I_Build

  Wave-I War Room — truth-first operator dashboard for **Blue Bucket** (BDC universe, primary: ARCC) and **Green Bucket** (mREIT universe, primary: AGNC).

  ## Features
  - Live scrolling market tape (17+ symbols)
  - Full-bucket quote board with dropdown selectors (20 BDCs / 17 mREITs)
  - Movement stats with 52W/day range bars
  - Yield / Dip / DRiP deterministic calculators
  - Research links panel
  - Portfolio tracker: add/view/edit/delete holdings with live G/L, dividends, DRiP, expected income — persisted in localStorage

  ## Stack
  React + Vite + TypeScript · Tailwind CSS · TanStack Query · Yahoo Finance (via corsproxy.io) · No backend needed
