# Wave-I Scope

Canonical scope for this branch:
- frontend-only
- truth-first
- provenance-explicit
- local-first
- static-deploy capable
- no runtime API dependence

Canonical object classes:
- [B] and [G] are buckets
- |W| and |M| are wallets
- |M| is nested inside [G]
- buckets and wallets stay distinct in model and UI

In scope:
- truth classes
- source classes
- provenance visibility
- local-only harvest
- quarantine-not-delete
- holdings and wallet semantics
- deterministic calculators
- local persistence

Out of scope:
- backend APIs
- remote persistence
- hidden cloud sync
- prediction engines
- recommendation engines
- smart-summary scoring
- future-wave automation
